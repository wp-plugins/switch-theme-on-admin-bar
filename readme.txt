== Description ==

This plugin adds Theme menus in the admin bar. 
You can switch and activate the theme on the admin bar. 
You don't need to go to the administration screens to switch the theme.

(For Japanese User)
管理バーにテーマメニューを追加します。
管理バー上でサイトテーマの変更・有効化が行えます。
テーマを変更するために管理画面に行く必要がなくなります。



== Installation ==

1. Upload the plugins to /wp-content/plugins/.
2. Activate the plugin through the 'Plugins' menu in WordPress

プラグインファイルを「/wp-content/plugins/」にアップロードします。
管理画面のプラグインメニューでプラグインを有効化します。
